# UI / Icon / Brand Rules

# Refab Connect / AI-WOC

## Visual direction

Premium, mobile-first, dark glass interface.

The app should feel like a clean operational tool, not a cluttered form dump.

## Brand principles

- Standardize to Optimize
- Built from the floor up
- Practical manufacturing intelligence
- Clear, controlled, shop-friendly workflow

## UI rules

- Mobile-first layout
- Dark premium background
- Subtle glass cards
- Strong readable text contrast
- Clear bottom navigation when workflow screens are added
- Large touch targets
- Simple forms with clear required fields
- Confirmation-before-send must be visually obvious

## Icon rules

- App icons belong in `public/icons`
- Logo files belong in `public/logos`
- Supporting images belong in `public/images`

## Styling ownership

- Color and spacing tokens: `styles/tokens.css`
- Base HTML/body rules: `styles/base.css`
- Layout structure: `styles/layout.css`
- Cards, buttons, inputs, chips: `styles/components.css`

## UI migration rule

Do not rebuild every screen at once.

Stabilize one screen or component group per Codex ticket.
